# Fichiers audio ClearBus

Ajoutez vos fichiers MP3 ici. Le site fonctionne sans eux, mais la radio et les annonces seront silencieuses.

## Structure requise

### Effet sonore (avant chaque annonce)
- `sfx/chime.mp3` — son type métro / « tududut »

### Radio (musique libre de droits)
- `music/track1.mp3`
- `music/track2.mp3`
- `music/track3.mp3`
